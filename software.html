Software¶

X-ray Absorption Spectroscopy software packages¶

The following all have some sort of web presence, not all of which are exactly up-to-date. This list is presented alphabetically by package name.

  • CDXAS : Classic and Dispersive X-ray Absorption Spectroscopy

  • Demeter : Bruce Ravel’s collection of analysis programs (Athena, Artemis, and Hephaestus)

  • DL_Excurv : Later version of EXCURV98

  • EDA : EXAFS Data Analysis Software Package (EDA) from Alexai Kuzmin

  • EXAFSPAK from Graham George

  • FDMNES from Yves Joly

  • FEFF

  • FitIt

  • GNXAS

  • Larch : Open-source library and set of applications for processing and analyzing X-ray absorption and fluorescence spectroscopy data and X-ray fluorescence and diffraction image data from synchrotron beamlines.

  • LASE : Logiciel d’Analyse des Spectres EXAFS

  • NPI and XAID

  • RSXAP : EXAFS Analysis Package from Corwin Booth

  • SEDEM

  • SixPACK : Sam’s Interface for XAS Package, a software package for x-ray absorption spectroscopy data

  • TT-Multiplets

  • Viper and Xanda by Konstantin Klementiev

  • WinXAS by Thorsten Ressler

  • XDAP for Windows from XAFS Services International

  • XANADU by Masao Takahashi and Hideto Sakane
